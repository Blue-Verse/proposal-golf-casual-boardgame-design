핵심 요구사항 분석

  • 아이디어 구체화 — 라운딩 순서 정하기, 경품 추천 이벤트 게임 등 초기 아이디어를 상세 기획으로 발전
  • 요구사항 정의 — 게임 참여 조건, 진행 규칙, 결과 처리 등 기능 요구사항 명세
  • 화면 설계 — 게임 참여·진행·결과 확인 등 관련 화면 일체의 와이어프레임
  • 정책 설계 — 게임 이코노미, 보상 체계, 경품 룰 등 서비스 정책
  • UI/UX 디자인 — Figma를 활용한 Android/iOS 앱 디자인 산출물
  • 산출물 — 요구사항 정의서, 기능 명세서, 화면 설계서, Figma 디자인 원본

제안 포인트

본 프로젝트에 다음과 같은 강점을 바탕으로 최적의 결과물을 제공할 수 있습니다:

  • 캐주얼 게임 기획·개발 경험 — 텔레그램 기반 캐주얼 게임 프로젝트에서 게임 로직 설계, 리더보드, 리퍼럴 시스템 등 게임 이코노미 전반을 기획·구현한 경험이 있습니다.
  • 게이미피케이션 UX 설계 역량 — 운세 앱, Sleep-to-Earn 앱 등에서 포인트/리워드, 데일리 체크인, 루트박스 시스템 등 게이미피케이션 요소를 설계하고 구현한 경험이 있습니다.
  • 모바일 앱 UI/UX 전문성 — Android/iOS 크로스플랫폼 앱 다수 수행 경험으로, 모바일 환경에 최적화된 직관적인 게임 UX를 설계할 수 있습니다.
  • 기획~디자인 일관성 — 기획자와 디자이너를 겸임하므로 기획 의도가 디자인까지 일관되게 반영됩니다. 커뮤니케이션 로스 없이 빠르고 정확한 진행이 가능합니다.

PRD (Product Requirements Document)

골프 앱 내 캐주얼 보드게임의 기능 범위와 요구사항을 정의합니다.

1. 프로젝트 배경 및 목적

현재 골프장 결제 서비스를 제공하는 앱을 운영 중이며, 골퍼들이 골프장에서 간단히 즐길 수 있는 캐주얼 보드게임을 추가하여 앱 이용률과 체류 시간을 증가시키는 것이 목표입니다. 초기 아이디어 단계에서 상세 기획과 UI/UX 디자인까지 구체화하는 프로젝트입니다.

2. 사용자 정의

사용자 유형설명주요 기능
골퍼 (일반 사용자)골프장에서 라운딩 중 게임을 즐기는 앱 사용자게임 참여, 주사위 굴리기, 결과 확인, 경품 받기
골프 그룹 호스트라운딩 그룹을 생성하고 게임을 시작하는 사용자그룹 생성, 게임 시작, 참여자 초대, 결과 공유
관리자게임 설정 및 경품 관리를 담당하는 운영자게임 설정, 경품 등록/관리, 통계 확인

3. 기능 요구사항

3.1 라운딩 순서 정하기 게임

  • [필수] 주사위 굴리기 — 각 골퍼가 주사위를 굴려 라운딩 순서 결정
  • [필수] 순서 결과 화면 — 주사위 결과 기반 순서 자동 정렬 및 표시
  • [필수] 동점 처리 — 같은 숫자가 나온 경우 재도전 로직
  • [선택] 결과 공유 — 순서 결과를 카카오톡 등으로 공유

3.2 경품 추천 이벤트 게임

  • [필수] 보드판 구성 — 경품이 배치된 보드판 화면
  • [필수] 주사위 기반 이동 — 주사위 굴림에 따라 보드판 이동
  • [필수] 경품 당첨 화면 — 도착한 칸의 경품 안내 및 당첨 연출
  • [필수] 경품 목록 조회 — 획득한 경품 이력 확인
  • [선택] 경품 수령 가이드 — 경품 교환 방법 안내

3.3 공통 기능

  • [필수] 게임 로비 — 게임 선택 및 참여 화면
  • [필수] 그룹 생성/참여 — QR코드 또는 코드 입력으로 같은 게임방 참여
  • [필수] 게임 결과 히스토리 — 과거 게임 결과 조회
  • [선택] 애니메이션 가이드 — 주사위 굴림, 말 이동 등 인터랙션 연출 가이드

4. 비기능 요구사항

항목요구사항
플랫폼Android/iOS 앱 내 WebView 또는 네이티브 컴포넌트
접근성모바일 환경 최적화, 큰 터치 영역, 직관적 인터랙션
디자인 일관성기존 골프 앱 디자인 시스템과의 조화
UX골프장 현장에서 빠르게 조작 가능한 간결한 플로우

5. 기술 제약사항

  • 기존 골프 앱 내에 임베딩되는 형태 — 앱의 기존 디자인 시스템과 조화 필요
  • 디자인 산출물은 Figma 기반으로 제작
  • 개발팀(1명)과의 원활한 핸드오프를 위한 디자인 가이드 포함

6. 범위 외 사항 (Out of Scope)

  • 게임 로직 개발 (프론트엔드/백엔드)
  • 서버 인프라 구축
  • 실제 경품 물류/배송 시스템
  • 결제 시스템 연동
  • 앱 스토어 배포

프로젝트 개요

텔레그램 기반 캐주얼 아케이드 게임 기획·설계·개발. Phaser.js 게임 엔진을 활용하여 간단하고 중독성 있는 게임 플레이를 구현하고, 리더보드·리퍼럴 시스템 등 게임 이코노미를 설계했습니다.

핵심 기능

  • Phaser.js 기반 아케이드 게임 (터치 인터랙션)
  • 리더보드 · 랭킹 시스템
  • 2단계 리퍼럴 시스템
  • 게임 이코노미 설계 (포인트, 보상)

본 프로젝트와의 유사점

캐주얼 게임 기획·설계 경험이 가장 직접적으로 매칭됩니다. 게임 룰 설계, 보상 시스템, 리더보드, 사용자 참여 유도 메커니즘 등 본 프로젝트에서 요구하는 게임 이코노미 설계 역량을 갖추고 있습니다.

기술 스택

Phaser.js JavaScript NestJS Docker

사주 기반 운세 분석, 행운의 번호 추천, 궁합 분석 기능을 제공하는 크로스플랫폼 모바일 앱. 기획부터 디자인, 개발까지 전 과정을 수행했습니다.

핵심 기능

  • Android/iOS 동시 출시

본 프로젝트와의 유사점

모바일 앱 UX 기획·디자인과 게이미피케이션 요소 설계 경험이 유사합니다. 포인트/리워드 시스템, 데일리 체크인, 뽑기(가챠) 요소 등 사용자 참여를 유도하는 게이밍 메커니즘을 기획하고 Figma로 디자인한 경험이 있습니다.
